# Opps default backend

Simple go & chi based default backend for handling errors and not founds. 

Default template and errors definitions included.
If you want to customize the error pages you can mount a template and errors definition yaml.

`template.tmlp mount to /template.tmpl`
`errors.yaml mount to /errors.yaml`

you can use a prebuilt image available on `quay.io/mkaplan/opps:latest`
